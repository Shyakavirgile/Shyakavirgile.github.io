5 HTML Tags

  • title: Shows the document's title that is shown in a page's tab.
  • h1: It is commonly used to display a web page title.
  • h2: It is used to mark the first sub-heading after your document's main heading.
  • body: It is used to define the document's body.
  • html: It is used to show the beginning and end of an HTML element in an HTML document.
